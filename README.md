Studi Kasus: Sistem Pendaftaran Kursus Online
Deskripsi Masalah:

Sebuah platform kursus online ingin membuat sistem pendaftaran sederhana. 
Data yang perlu ditangani mencakup informasi peserta, kursus yang diambil, dan status pendaftaran. 
Kamu diminta untuk menyusun representasi data menggunakan tipe data Python yang tepat.

Case Study: Online Course Registration System
Problem Description:

An online course platform wants to create a simple registration system.
The data that needs to be managed includes participant information, courses taken, and registration status.
You are asked to design a data representation using appropriate Python data types.
